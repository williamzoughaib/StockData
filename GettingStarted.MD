# Getting Started

## Prerequisites

- Anaconda or Miniconda installed
- Python 3.12 (managed via conda)

###
- NVIDIA GPU with CUDA 12.1 support (optional, for PyTorch GPU acceleration). Can replace with CPU acceleration. 
- NVIDIA GPU with RAPIDS acceleration. Requires Linux distro. Tested on Fedora Gnome. 

## Environment Setup

This project uses **Python 3.12** with conda. 
- `environment.yml` - Conda packages
- `environment-gpu.yml` - Conda packages with RAPIDS GPU acceleration 
- `requirements.txt` - Pip packages
